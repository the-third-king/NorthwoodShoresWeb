  4. We have a navigation bar on the left of every page

  6. We mainly use css cards and the same nav bar and header on every page so it looks coherent.
  7. All images have an alt category.
  8. Every page has the same campfire favicon since it is a campground
  9. We have 2 folders for css and images and leave the html out in the parent folder for easy access to get to.
  10. We only used css and html. There is a javascript file in here but that method is commented out for the time being.
  11. Every page passes the html and css validator.
  12. We have a github at Northwood Shores Github.
  13. All files are lowercase using alphanumeric letters only.
  14. All content is aligned in there own way to make pages look cohesive.
  15. We have 2 different font faces for the pages and for the header bar.
  16. we have links to external sites in the area attractions page and also a link to facebook in the footer.
  17. The source code is readable using understandable names.
  18. The nav bar is the same on every page and even when shrunk down it is consistent.
  19. Every webpage has a header and footer section with the navbar.
  20. We use mainly flexboxes in all of our pages.
